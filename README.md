# Neo Snake Game
Snake Game But in a Console.

## About
This is a Snake Game made entirely by myself. It is made in Entirely in C++.
This Game helped me a lot in improving my C++ Skill. Object Oriented Programming to be Exact.

## Features
This Game has Different Difficulty Levels you can set the Difficulty in-game. As well as edit able Player name, Which is Entirely Change Able After Each Round.
Touching Any Border of the Board will kill the snake.

## How to Play
Use W , A , S , D to Move the snake around the game. Try to Score the Highest!

## Difficulty Levels
The Game Speed are:
1. Easy = 200ms Delay.
2. Medium = 100ms Delay.
3. Hard = 50ms Delay.
4. Hardcore = 25ms Delay.
   This "Delay" is the in-game delay of the main game loop.
